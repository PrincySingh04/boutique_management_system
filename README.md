# Boutique Management System

A Python-based Boutique Management System designed to manage:
- Customer details
- Product records
- Orders and billing
- Database connectivity using MySQL

## Technologies Used
- Python
- MySQL
- MySQL Connector
- xampp

## Features
- Customer registration
- Product management
- Order handling
- Secure database operations

## How to Run
1. Install Python
2. Install MySQL
3. Configure database credentials
4. Run the Python file
